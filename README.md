# ___Comment ça marche___ :robot:

Pour démarrer l'application, lancez les commandes suivantes :
```
bundle install
rails db:create
rails db:migrate
rails db:seed
rails server
```

Puis rendez-vous sur votre navigateur pour acceder au site (http://localhost:3000)
